Welcome to KLIPPER board flashing

This site will guide you how to flash KLIPPER firmware to your board.

Building firmware

1.) Connect to your klipper server and update the system

sudo apt update && sudo apt upgrade -y
Copied!

2.) Build firmware

> move to the klipper directory
cd ~/klipper
Copied!
> open menuconfig
make menuconfig
Copied!
> setup menuconfig
Select your Micro-controller
- press Q key and save configuration with Y key
> compile firmware
make
Copied!

Flashing firmware

1.) Enquire board ID

ls /dev/serial/by-id/*
Copied!
Now you should see something like /dev/serial/by-id/usb-1a86_USB2.0-Serial-if00-port0, every board has it's unique ID.

2.) Flash firmware

> stop klipper
sudo service klipper stop
Copied!
> flash firmware
cd ~/klipper/ && make flash FLASH_DEVICE=/your board id/
Copied!
> start klipper
sudo service klipper start
Copied!
